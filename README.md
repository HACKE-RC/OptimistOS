# OptimistOS 
UEFI Based OS for x86_64 which I made for fun.

# Features

## Bootloader
- [x] [Limine](https://github.com/limine-bootloader/limine)

## CPU
- [x] GDT
- [x] IDT
- [x] ACPI
- [x] SMP
- [x] Scheduling

## Memory
- [x] Virtual Memory
- [X] Heap using [mallocx](https://github.com/HACKE-RC/mallocx)
